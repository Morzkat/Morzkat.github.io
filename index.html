Trinity Web App

Web App para el manejo de estaciones de combustibles donde podrá ver: las ventas, cuadrar, realizar depósitos, pagos, tener clientes de créditos, también tendrá un Dashboard donde podrá observar la cantidad de ventas diarias, semanales y mensuales, tendrá comparativas para el desempeño de la estación, contará con varios tipos de reportes como son por turno, período entre otros, este sistema cuenta con muchas más funcionalidades, para obtener más información del mismo, puede solicitarlo en el link proporcionado!

Tecnologías utilizadas:

Este sistema esta construido en Asp.net 4.5 para el manejo del Back-End y Angular 4 para el manejo del Front-End también se utiliza Bootstrap 4 para la interfaz de la App, los reportes son manejados usando Crystal Reports, también hay servicios de Windows los cuales se encargan de migrar la data para que todos los otros procesos sean mas ágiles.

Dadga Mobile App

La aplicación del entrenador Francisco Alcala fue desarrollada con la intencion que sus clientes lleven un registro de sus actividades físicas y alimenticias y todas estas actividades su entrenador las pueda visualizar, también por la App son asignadas las rutinas a seguir que constan de una serie de vídeos y instrucciones, la subscripción y ofertas son manejada desde la misma App facilitando simplificando la vida de los clientes.

Tecnologías utilizadas:

Dadga Mobile App esta hecha en React Native, es una aplicación que esta disponible tanto en IOS como Android, esta aplicación tiene un Back-End que esta hecho en .Net Core 2.0 y esta alojado en los servidores de Heroku.

Para usar esta aplicación debes comunicarte con el dueño de esta.

Sise

Junto al "Team Kakaroto" estoy desarrollando esta aplicación, Esta aplicación tiene como fin llevar un registro de fincas y productores que hay en la República Dominicana, para poder realizar un distribución correcta de los recursos que son dados a los productores y tener toda esta información centralizada, esta aplicación esta siendo supervisada y dirigida por el ministerio de agricultura.

Tecnologías utilizadas:

Esta aplicación utiliza para el Back-End PHP (el framework de Laravel) y ReactJS para el Front-End, para guardar los datos se esta utilizando la base de datos de MySql.
